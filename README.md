# AI Tester Project

## 📌 Description
Hệ thống sinh và thực thi test case tự động bằng AI (OpenAI + Selenium + Pytest).

## 🚀 Features
- Tự động sinh test case bằng AI
- Chạy test bằng Selenium
- Đánh giá kết quả

## 🛠️ Tech stack
- Python
- Selenium
- Pytest
- OpenAI API

## ▶️ How to run
```bash
pip install -r requirements.txt
python main.py

## 🔄 Workflow
1. Input: URL website
2. AI sinh test case
3. Selenium thực thi
4. Pytest đánh giá

## 📊 Example Output
- Test case: login button clickable
- Result: PASS / FAIL

## 🎯 Result
- Số test case sinh ra
- Tỉ lệ pass
